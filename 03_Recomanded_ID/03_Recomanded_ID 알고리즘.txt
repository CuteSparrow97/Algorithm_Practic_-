03_Recomanded_ID
1. 카카오 ID 규칙 : 3글자이상 15자 이하
2. 아이디는  알파베 소문자, 숫자, 빼기(-),밑줄(_), 마침표(.) 사용
3. 단, 마침표(.)는 처음과 끝에 사용 불가능, 또한 연속 사용 불가능

7단계 순차적 처리 과정
1단계 new_id의 모든 대문자를 대응되는 소문자로 치환합니다.
2단계 new_id에서 알파벳 소문자, 숫자, 빼기(-), 밑줄(_), 마침표(.)를 제외한 모든 문자를 제거
3단계 new_id에서 마침표(.)가 2번 이상 연속된 부분을 하나의 마침표(.)로 치환합니다
4단계 new_id에서 마침표(.)가 처음이나 끝에 위치한다면 제거
5단계 new_id가 빈 문자열이라면, new_id에 "a"를 대입
6단계 new_id의 길이가 16글자 이상이면, new_id의 첫 15개의 문자를 제외한 나머지 문자들을 모두 제거합니다.
		만약 제거 후, 마침표가(.)가 new_id 끝에 위치한다면 끝에 있는 마침표 제거
7단계 new_id의 길이가 2자 이하라면, new_id의 마지막 문자를 new_id의 길이가 3이 될때까지 반복해서 끝에 붙입니다.

제한사항
new_id는 길이 1이상 1,000 이하인 문자열입니다.
특수문자 : -_.~!@#$%^&*()=+[{}]:?,<>/ 로 한정됩니다.

입력 string new_id
출력 string answer
